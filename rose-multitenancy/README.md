# Rose Multitenancy

Tenant context and web integration for Rose applications.

| Module | Artifact | Role |
|--------|----------|------|
| Core | `rose-multitenancy-core` | Tenant context APIs |
| Web | `rose-multitenancy-web` | Servlet/WebMVC integration |
| Starters | `rose-multitenancy-core-spring-boot-starter`, `rose-multitenancy-web-spring-boot-starter` | Application entry points |
